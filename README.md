# Airbnb Clone

A simple Airbnb-like app for listing and booking properties.

## Features

- User login/signup
- Browse listings
- View details
- Add new listings

## Installation

1. Clone repo
2. Install dependencies
3. Run server
4. Visit localhost

## Technologies

- HTML, CSS, JS, Bootstrap
- Node.js / Express (or Django)
- MongoDB / SQL
